# Cloud Computing

- Cloud framework
	- Containing both a physical layer and virtualization layer
	- Physical
		- Equipment assets
		- Servers
		- System parts
	- Virtual
- Cloud computing
	- A space over network infrastructure where computing resources are available instantly, on demand
		- Hardware, storage, databases, networks, OS, and entire applications
	- Applications
		- Scalability
		- Cost savings

- Cloud computing is compared with service oriented architectures (SOA), Grid computing, Utility computing, and Cluster computing
- Persued independently
- Cloud platforms can be added to SOA
- Grid computing, and computing resources can be provisioned as utility
- Cloud computing
	- On-demand resource provisioning
	- Removes over-provisioning to accomodate the demands of several customers

- Cluster computing
	- Cheaper-cost form for processing application that run in parallel
	- End users access Cloud services and data centre resources via the internet
	- Cloud service provider (CSP) provisions resources in the form of VM to users
	- Users rend OS, CPU, memory, storage, and network resources
