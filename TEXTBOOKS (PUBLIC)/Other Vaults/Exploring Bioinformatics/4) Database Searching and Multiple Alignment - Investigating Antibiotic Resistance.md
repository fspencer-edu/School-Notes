## Understanding the Algorithm: Database Searching and Multiple Alignment

**BLAST: A Heuristic Approach to Database Searching**

- Needle-Wunsch algorithm is a relatively efficient algorithm fr optimal, global pairwise sequence alignment
	- $O(NS)$
	- S = number of sequences
- BLAST (Basic Local Alignment Search Tool)
	- Compare a query sequence to the entire database and return all matching sequences
	- Use heuristic that makes practical and efficient solution
- Break the query sequence into short world called $k$-tuples
- Default setting is 11 for DNA and 3 for protein alignments
- BLAST scores close matches between short sequences and each database called seeding
- When there is a good match, a local alignment is used to extend the alignment in both directions
- Comparing the resulting score with a threshold value
	- High-scoring pair (HSP)

- BLAST then calculates the statistical likelihood that a given score would occur based on chance alignment of unrelated sequences (e-value)
- Then orders this statistical significance
- Decreasing the word size allows the identification of sequences of sequences that match less well
- Heuristics approach allows to quickly discriminate possible matches from unrelated sequences
- Helps deal the large volumes of data

**ClustalW: Multiple Sequence Alignment**
- Displays only individual alignment of the query with each matching sequence
- Consensus sequence
	- Nucleotides or amino acids that appear the most frequently at each position
- Multiple sequence alignment
	- Pairwise alignment
- Computational time of Needle-Wunsch becomes $O(S^2)$ for large values
- ClustalW is one of the most popular multiple sequence alignment algorithms
	- Progressive alignment
		- Order of adding new sequences to the alignment is determined by calculating a phylogenetic tree (guide tree)
- Guide tree is generated by pairwise alignments
- ClustalW does global, pairwise alignments to align, in order of decreasing relatedness

**Antibiotic Resistance and Gene Transfer**
- Bacteria can have natural (intrinsic) resistance to some antibiotics because of cell structure
	- Gram-negative is resistant to penicillin
- Resistance that is important medically is acquired resistance
	- Reduced resistance from antibiotic use
- Antibiotics can select for bacteria that already become resistant, allowing them to become more prevalent
	- Do not finish antibiotic prescription
- Cell's descendants inherit from vertical gene transfer
	- Lead to increased resistance by selection if the population is challenged by antibiotic
		- Mutations are more rare and slow
- Antibiotic resistance is carried by
	- Plasmids
		- Small, circular, independent DNA molecules
- Transfer would occurs by conjugation (cell-to-cell contact) by bacteria virus (transduction), or by direct uptake of DNA into the environment (transformation)
