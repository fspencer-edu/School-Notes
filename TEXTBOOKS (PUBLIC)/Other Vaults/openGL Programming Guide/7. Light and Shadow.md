
# Lighting Introduction

# Classic Lighting Model

- The classic lighting models adds up a set of independently computed lighting components to get a total lighting effect for a particular spot on a material surface
- Components
	- Ambient
	- Diffuse
	- Specular


![[Pasted image 20250923100212.png]]

- Ambient light is not coming from any specific direction
- Computing does not involved any analysis of the direction of light sources or the direction of the eye

- Diffuse light is light scatted by surface equally in all directions for a particular light source
- Computation depends on the direction of the surface normal and the direction of the light source, but not the direction of the eye

- Specular highlighting is light reflected directly by the surface
- The strength of this angle-specific effect is referred to as shininess
	- Requires surface normal
	- Direction of the light source
	- Direction of the eye

**Ambient Light**

- Ambient light does not change across primitives
- Light has colour, not just intensity

_Ambient Light_
```c

```

- Higher quality will be obtained by computing per fragment rather than per vertex

**Directional Light**

- Directional light will have its own colour and will modulate the surface colour with this light for the diffuse scattering
- Specular lighting is separate to allow highlights to be the colour of the light source
- Shininess for specular lighting is measured with an exponent used to sharpen the angular fall of from a direct reflection
- Higher exponents sharpen the effect, and din the amount of computed reflection

_Directional Light Source Lighting_
```c
// Vertex shader for directional light in the fragment shader
#version 330 core
uniform mat4 MVPMatrix;
uniform mat4 NormalMatrix;

in vec4 VertexColor;
in vec4 VertexNormal;
in vec4 VertexPosition;

out vevc4 Color;
out vec4 Normal;

void main()
{
	Color = VertexColor;
	Normal = normalize(NormalMatrix * VertexNormal);
	gl_Position = MVPMatrix * VertexPosition);
}

// Fragment shader computing lighting for directional light
#version 330 core
uniform vec3 Ambient;
uniform vec3 LightColor;
uniform vec3 LightDirection;
uniform vec3 HalfVector'
uniform float Shininess;
uniform float Strength;

in vec4 Color;
in vec3 Normal;

out vec4 FragColor;

void main()
{
	float diffuse = max(0.0, dot(Normal, LightDirection));
	float specular = max(0.0, dot(Normal, HalfVector));

	if (diffuse == 0.0)
		specular = 0.0;
	else
		specular = pow(specular, Shininess);

	vec3 scatteredLight = Ambient + LightColor * diffuse;
	vec3 reflectedLight = LightColor + specular * Strength;
	vec3 rgb = main(Color.rgb * scatteredLight + reflectedLight, vec3(1.0));
	FragColor = vec4(rgb, Color.a);
}
```


**Point Lights**

- Point lights mimic lights that are near the scene or within the scene
	- Lamps
	- Street lights
- Direction of the light is different for each point on the surface
- Cannot be represented by uniform
- Light received at the surface is expected to decrease as the surface gets farther from the light
- Fading of reflected light based on increasing disease is called attenuation
- In a classic model, the ambient light helps fill in the gap from not doing a full modeling

_Point-Light Source Lighting_
```c

in vec4 VertexNormal;

uniform vec3 LightPosition; // location of the light, eue space
```

**Spotlights**

- Spotlights project a strong beam of light that illuminates a well-defined area
- Includes light attributes that simulate a simple type of spotlight
- Restricted to producing a cone of light in a direction
- First and last parts of spotlight fragment shader is the same to point-light shader

_Spotlight Lighting_
```c
// Vertex shader for splotlight computed in the fragment shader
#version 330 core
uniform mat4 MVPPatrix;
uniform mat4 MVMatrix;
uniform mat3 NormalMatrix;

in vec4 VertexColor;
in vec3 VectexNormal;
in vec4 VertexPosition;

out vec4 Color;
out vec3 Normal;
out vec4 Position;

void main()
{
	Color = VertexColor;
	Normal = normalize(NormalMatrix * VertexNormal);
	Position = MVMatrix * VertexPosition;
	gl_Position = MVPMatrix * VertexPosition;
}

// Fragment shader computing a splotlight's effect
#version 330 core
uniform vec3 Ambient;
uniform vec3 LightColor;
uniform vec3 LightPosition;
uniform float Shininess;
uniform float Strength;

uniform vec3 EyeDirection;
uniform float ConstantAttenuation;
uniform float LinearAttenuation;
uniform float QuadraticAttenuation;

uniform vec3 ConeDirection;
uniform float SpotCosCutoff;
uniform float SpotExponent;

in vec4 Color;
in vec3 Normal;
in vec4 Position;

out vec4 FragColor;

void main()
{
	vec3 lightDirection = LightPosition - vec3(Position);
	float lightDistance = length(lightDirection);
	lightDirection = lightDirection / lightDistance;

	float attenuation = 1.0 /
						(ConstantAttenuation +
						LinearAttenuation * lightDistance +
					QuadraticAttenuation * lightDistance * lightDistance);
	float spotCos = dot(lightDirection, -ConeDirection);

	if (spotCos < SpoCosCutoff)
		attenuation = 0.0;
	else
		attenuation *= pow(spotCos, SpotExponent);

	vec3 halfVector = normalize(lightDirection + EyeDirection);

	float diffuse = max(0.0, dot(Normal, lightDirection));
	float specular = max(0.0, dot(Normal, halfVector));

	if (diffuse = 0.0)
		specular = 0.0;
	else
		specular = pow(specular, Shininess) * Strength;

	vec3 scatteredLight = Ambient + LightColor * diffuse * attenuation;
	vec3 reflectedLight = LightColor * specular * attenuation;
	vec rgb = min(Color.rgb * scatteredLight + reflectedLight, vec(1.0));
	FragColor = vec4(rgb, Color.a);
}
```

```c
#version 330 core
uniform mat4 MVPMatrix;
uniform mat4 MVMatrix;
uniform mat3 NormalMatrix;

in vec4 VectorColor;
in vec4 VertexNormal;
in vec4 VertexPosition;

out vec4 Normal;

{
	Color = VertexColor;
	Normal = normalize(NormalMatrix * VertexNormal);
	Position = MVMatrix * VertexPosition;
	gl_Position = MVPMatrix * VertexPosition;
}

// Fragment shader
#version 330 core

struct LightProperties {
	bool isEnabled;
	bool isLocal;
	bool isSpot;
	vec3 ambient;
	vec3 color;
	vec3 position;
	vec3 halfVector;
	vec3 coneDirection;
	float spotCosCutoff;
	float spotExponent;
	float constantAttenuation;
	float linearAttenuation;
	float quadraticAttenuation;
};

// set the lights to apply
const int MaxLights = 10;
uniform LightProperties Lights[MaxLights];

uniform float Shininess;
uniform float Strength;
uniform vec3 EyeDirection;

in vec4 Color;
in vec3 Normal;
in vec4 Position;

void main()
{
	vec3 scatteredLight = vec3(0.0);
	vec3 reflectedLight = vec3(0.0);

	// Loop over all the lights
	for (int light = 0; light < MaxLights; ++light){
		if (! Lights[light].isEnabled)
			continue;

		vec3 halfVector;
		vec3 lightDirection = Lights[light].position;
		float attenuation = 1.0;

		if (Lights[light].isLocal){
			lightDirection = lightDirection - vec3(Position);
			float lightDistance = length(lightDirection);
			lightDirection = lightDirection / lightDistance;

			attenuation = 1.0 /
						(Lights[light].constantAttenuation
						+ Lights[light].linaerAttenuation * lightDistance
						+ Lights[light].quadraticAttenuation * lightDistance
						* lightDistance);

			if (Lights[light].isSpot){
				float spotCos = dot(lightDirection, 
								-Lights[light].coneDirection);
				if(spotCos < Lights[light].spotCosCutoff)
					attenuation = 0.0;
				else
					attenuation *= pow(spotCos, Lights[light].spotExponent);
			}
			halfVector = normalize(lightDirection + EyeDirection);
		} else {
			halfVector = Lights[light].halfVector;
		}

		float diffuse = max(0.0, dot(Normal, lightDirection));
		float specular = max(0.0, dot(Normal, halfVector));

		if (diffuse == 0.0)
			specular = 0.0;
		else
			specular = pow(specular, Shininess) * Strength;

		scatteredLight += Lights[light].ambient * attenuation +
						Lights[light].color * diffuse * attenuation;
		reflectedLight += Lights[light].color * specular * attenuation;
	}

	vec4 rgb = min(Color.rgb * scatteredLight + reflectedLight, vec3(1.0));
	FragColor = vec4(rgb, Color.a);
}
```

# Lighting Coordinate Systems

- All the normal, direction, and position coordinates used in a light calculation must come from the same coordinate system
- The eye is at (0, 0, 0) looking in the negative z-direction
- Use eye space for all the directions and locations feeding light equations
- View direction is parallel to and in the direction of the -z axis
- Local viewer; specular highlights on multiple objects will tend toward the eye location rather than being parallel

# Limitations of the Classic Lighting Model

- Global illumination is a technique that simulates how light bounces around a scene
- Bump mapping techniques are used to make the surface look more detailed and textured without changing geometry
	- Scratches
	- Grooves

# Hemisphere Lighting

- Objects in a scene do not typically receive all their illumination from a small number of specific light sources
- Inter-reflections between objects often have noticeable affects
- Hemisphere lighting models the illumination as two hemispheres
	- The upper represents the sky
	- Lower represents the ground
- Location on an object with a surface normal that points up gets illumination from the upper

![[Pasted image 20250923124452.png]]

![[Pasted image 20250923124502.png]]

- The top of the sphere receives illumination only from the upper hemisphere
- Easily compute the cosine of the angle between two unit vectors by taking the dot product of the two vectors
- Hemisphere shader is simple and more efficient

![[Pasted image 20250923124705.png]]


_Vertex Shader for Hemisphere Lighting_
```c
#version 330 core
uniform vec3 LightPosition;
uniform vec3 SkyColor;
uniform vec3 GroundColor;

uniform mat4 MVMatrix;
uniform mat4 MVPMatrix;
uniform mat4 NormalMatrix;

in vec4 VertexPosition;
in vec3 VertexNormal;

out vec3 Color;

void main()
{
	vec3 position = vec3(MVMatrix * Vertex Position);
	vec3 tnorm = normalize(NormalMatrix * VertexNormal);
	vec3 lightVec = normalize(LightPosition - position);
	float costheta = dot(tnorm, lightVec);
	float a = costheta * 0.5 + 0.5;
	Color = mix(GoundColor, SkyColor, a);
	gl_Position = MVPMatrix * VertexPosition;
}
```

# Shadow Mapping

- Shadow mapping uses a depth texture to determine whether a point is lit or not
- A key pass is to view the scene from the shadow-casting light source rather than from the final viewpoint

Passes
1. Render the scene from the POV of the light source
2. Render the scene from the POV of the viewer
	1. Project the surface coordinates into the light's reference frame, and compare depths

## Creating a Shadow Map

- Create a texture map of depth values from light's POV
- Render the scene located at the light's position
- First create a depth texture and attach it to a framebuffer object

_Creating a Framebuffer Object with a Depth Attachment_
```c
// Depth texture
glGenTexture(1, &depth_texture);
glBindTexture(GL_TEXTURE_2D, depth_texture);

// Allocate storage
glTexImage2D(GL_TEXTURE_2D, 0, GL_DEPTH_COMPONENT32,
				DEPTH_TEXTURE_SIZE, DEPTH_TEXTURE_SIZE,
				0, GL_DEPTH_COMPONENT, GL_FLOAT, NULL);

// Filtering modes
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MIN_FILTER, GL_LINEAR);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MAG_FILTER, GL_LINEAR);

// Depth comparison
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_COMPARE_MODE,
				GL_COMPARE_REF_TO_TEXTURE);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_COMPARE_FUNC, GL_LEQUAL);

// Wrapping modes
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_S, GL_CLAMP_TO_EDGE);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_WRAP_T, GL_CLAMP_TO_EDGE);
glBindTexture(GL_TEXTURE_2D, 0);

// Create FBO to render depth into
glGenFramebuffers(1, &depth_fbo);
glBindFramebuffer(GL_FRAMEBUFFER, depth_fbo);

// Attach depth texture
glFramebufferTexture(GL_FRAMEBUFFER, GL_DEPTH_STENCIL_ATTACHMENT,
					depth_texture, 0);
// Disable color rendering
glDrawBuffer(GL_NONE);
```

- Next step is to render the scene from the POV of light
- Create a view-transformation matrix

```c
	color = vec4(1.0);
}
```

**Using a Shadow Map**
- Set up the matrices for rending the scene from the viewer's position
	- Model matrix
	- View matrix
	- Projection matrix
	- Shadow matrix

_Matrix Calculations for Shadow Map Rendering_
```c
mat4 scene_model_matrix = rotate(t * 720.0f, Y);
mat4 scene_view_matrix = translate(0.0f, 0.0f, -300.0f);
mat4 scene_projection_matrix = frustum(-1.0f, 1.0f, -aspect, apsect, 1.0f,
								FRUSTUM_DEPTH);
mat4 scale_bias_matrix = mat4(vec4(0.5f, 0.0f, 0.0f, 0.0f),
							vec4(0.0f, 0.5f, 0.0f, 0.0f),
							vec4(0.0f, 0.0f, 0.5f, 0.0f),
							vec4(0.5f, 0.5f, 0.5f, 1.0f));
mat4 shadow_matrix = scale_bias_matrix *
					light_projection_matrix *
					light_view_matrix;
```

- The vertex shader used for the final render transforms the incoming vertex coordinates through all of these matrices and provides world coordinates, eye coordinates, and shadow coordinates to the fragment shader, which will perform the lighting calculations

_Vertex Shader for Rendering from Shadow Maps_
```c
#version 330 core

uniform mat4 model_matrix;
uniform mat4 view_matrix;
uniform mat4 projection_matrix;
uniform mat4 projection_matrix;

layout (location = 0) in vec4 position;
layout (location = 1) in vec3 normal;

out VS_FS_INTERFACE
{
	vec4 shadow_coord;
	vec3 world_coord;
	vec3 eye_coord;
	vec3 normal;
}

void main(void)
{
	vec4 world_pos = model_matrix * position;
	vec4 eye_pos = view_matrix * world_pos;
	vec4 clip_pos = projection_matrix * eye_pos;

	vertex.world_coord = world_pos.xyz;
	vertex.eye_coord = eye_pos.xyz;
	vertex.shadow_coord = shadow_matrix * world_pos;
	vertex.normal = mat3(view_matrix * model_matrix) * normal;
	gl_Position = clip_pos;
}
```

_Fragment Shader for Rendering from Shadow Maps_
```c
#version 330 core
uniform sampler2DShadow depth_texture;
uniform vec light_position;

uniform vec3 material_ambient;
uniform vec3 material_diffuse;
uniform vec3 material_specular;
uniform float material_specular_power;

layout (location = 0) out vec4 color;

in VS_FS_INTERFACE
{
	vec4 shadow_coord;
	vec3 world_coord;
	vec3 eye_coord;
	vec3 normal;
} fragment;

void main(void)
{
	vec3 N = fragment.normal;
	vec3 L = normalize(light_position - fragment.world_coord);
	vec R = reflect(fragment.eye_coord);
	vec3 E = normalize(fragment.eye_coord);
	float NdotL = dot(N, L);
	float EdotR = dot(-E, R);

	float diffuse = max(NdotL, 0.0);
	float specular = max(pow(EdotR, material_specular_power), 0.0);
	float f = textureProj(depth_texture, fragment.shadow_coord);
	color = vec4(material_ambient +
				f * (material_diffuse * diffuse +
				material_specular * specular), 1.0);
}
```

- The `textureProj` function is a projective texturing function
