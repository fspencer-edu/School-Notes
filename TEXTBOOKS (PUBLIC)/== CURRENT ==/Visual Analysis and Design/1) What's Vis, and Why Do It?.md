- Exploratory analysis for scientific discovery
	- Goal of speeding and improving a users's ability to generate and check hypotheses
- External representations
	- Also called external memory
	- Touchable physical objects
	- Augment human capacity to surpass the limitations of own internal cognition and memory
- Diagrams can be designed to support perceptual inferences

- A significant amount of visual information processing occurs in parallel at the preconscious level
	- Popout
- Interaction
	- Users actions cause the view to change 
	- Support investigation at multiple levels of detain
	- Represent and summarize data between multiple connections

- Vis idiom
	- A distinct approach to creating and manipulating visual representations
- Visual encoding
	- Process of mapping data into visual structures

- Satisfy
	- Find one of the many possible solutions rather than one of the even large number of bad ones
	- Opposed to optimize

- Validation
	- Many questions to consider for a vis tool to meet design goals
	- Difficult to benchmark data vis tools

- Limitations
	- Computational capacity

	- Computer time

	- Dataset size
- Soft constraint

- Change blindness
	- The phenomenon where even very large changes are not noticed if we are attending to something else in our view
- Display capacity
	- Resolution of the screen is not enough to show all desired information simultaneously
