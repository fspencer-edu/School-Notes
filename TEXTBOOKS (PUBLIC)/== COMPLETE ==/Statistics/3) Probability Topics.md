# Terminology

- Probability
	- A measure that is associated with how certain an outcome is to a particular experiment or activity
- Experiment
	- A planned operation carried out under controlled conditions
- Chance experiment
	- If the result is not predetermined
- Outcome
	- Result of an experiment
- Sample space, $S$
	- Set of all possible outcomes
		- List the possible outcomes
		- Create a tree diagram
		- Create a Venn diagram

$S = \{H, T\}$ => flip a fair coin

- Event
	- Any combination of outcomes
		- A, and B

$P(A)$ = probability of event A

- Probability
	- Long-term relative frequency of that outcome
	- Between zero and one, inclusive
- $P(A) = 0$
- Equality likely
	- Each outcome has a equal probability
- Probability of event A in a sample space where outcomes are equally likely
	- Number of outcomes divided by total number of outcomes in the sample space

- Law of large numbers
	- As the number of repetitions of an experiments increase, the relative frequency obtained in the experiment tends to become closer and closer and closer to the theoretical probability
- Empirical
	- Evidence through observations

- Unfair/biased
	- Outcomes are not equally likely

**OR Event**
- $A \cup B$

**AND Event**
- $A \cap B$

- Complete of event A, $A'$
	- All outcomes that are not in A
	- $P(A) = + P(A') = 1$

- Conditional probability
	- $P(A|B)$
	- A given B
	- A conditional reduces the sample space

$P(A|B) = P(A \cap B)/P(B)$

# Independent and Mutually Exclusive Events

- Independent and mutually exclusive do not mean the same thing

## Independent Events

- Two events are independent if the following are true
	- $P(A|B) = P(A)$
	- $P(B|A) = P(B)$
	- $P(A \cap B) = P(A)P(B)$

- Two events A and B are independent if the knowledge that one occurred does not affect the change the other occurs
- If events are not independent, then they are dependent events

- Sampling with replacement
	- Events are considered to be independent
- Sampling without replacement
	- Events are dependent

- Assume events are dependent until you can show otherwise
## Mutually Exclusive Events

- A and B are mutually exclusive events if they cannot occur at the same time
	- $P(A \cap B) = 0$

- Assume two events are mutually exclusive until you can show otherwise

# Two Basic Rules of Probability

## Multiplication Rule

- A and B are two events defined on a sample space
	- $P(A \cap B) = P(B)P(A|B)$

$P(A|B) = P(A \cap B)/P(A)$

- If A and B are independent, then 
	- $P(A|B) = P(A)$
	- $P(A \cap B) = P(B)P(B)$

## Addition Rule

- If A and B are defined on a sample space
	- $P(A \cup B) = P(A) + P(B) - P(A \cap B)$

- If A and B are mutually exclusive
	- $P(A \cap B) = 0$
	- $P(A \cup B) = P(A) + P(B)$
