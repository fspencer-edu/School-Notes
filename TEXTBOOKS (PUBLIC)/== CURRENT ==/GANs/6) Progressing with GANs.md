## Progressive growing and smoothing of higher-resolution layers

- Stochastic gradient descent
- Progressive growing
	- Increasing the resolution of the terrain as we go
- Progressively smooth in and slowly introduce complexity
	- Low-resolution convolutional layers to high-resolution

![[Pasted image 20260309094442.png]]

![[Pasted image 20260309094515.png]]

- When the 16 x 16 resolution has trained enough, a transposed convolution in the Generator is introduced
- Another convolution in the discriminator is added to get 32 x 32
	- $1- \alpha$ => nearest neighbour upscaling with not trained parameters
	- $\alpha$ => extra transposed convolution

- Smoothly fade into a new layer with higher resolution with $\alpha$ parameter between 0 and 1
- Alpha effects how much we use either old (upscaled) layer or the natively large one
- D downscales by 0.5x

## Mini-batch standard deviation

- Mini-batch standard deviation
- The standard deviation of all the pixels in the mini-batch are generated by the generator or come from the real data
	- Variation of samples
- Discriminator needs to learn that if the standard deviation is low in the images, the image is likely fake, since real data has more variance
	- Generator then produces samples with more variance
- Include a single extra number
	- Number is appended as a feature map

**Adding Std. D**
1. 4D -> 3D
	1. Compute standard deviation across all images in the batch (h, w, c)
		1. Get a single image with standard deviations for each pixel and each channel
2. 3D -> 2D
	1. Arrange the standard deviations across all channels, to get a single feature map/matrix for for that pixel, with a collapsed colour channel
3. 2D -> 0D
	1. Average the standard deviations for all pixels within the matrix to get a single scalar value

```python
def minibatch_std_layer(layer, group_size=4):
	group_size = K.backend.minimum(group_size, tf.shape(layer)[0])
	shape = list(K.int_shape(input))
	shape[0] = tf.shape(input)[0]
	
	minibatch = K.backend.reshape(layer,
		(group_size, -1, shape[1], shape[2], shape[3]))
	minibatch -= tf.reduce_mean(minibatch, axis=0, keepdims=True)
	minibatch = tf.reduce_mean(K.backend.square(minibatch), axis=0)
	minibatch = K.backend.square(minimatch + 1e8)
	minibatch = tf.reduce_mean(minibatch, axis=[1, 2, 3], keepdims=True)
	minibatch = K.backend.tile(minibatch,
		[group_size, 1, shape[2], shape[3]])
	return K.backend.concatenate([layer, minibatch], axis=1)
```

- Reshape the layer to that we operate on the level of the mini-batch
	- `[Group, mini-batch, width, height, channel]`
- Centres the mean over the group `[M, W, H, C]`
- Calculates the variance of the group `[M, W, H, C]`
- Calculates the standard deviation over`[M, W, H, C]`
- Takes the average over feature maps and pixels `[M, 1, 1, 1]`
- Transforms the scalar values to fit groups and pixels
- Appends as a new feature map

## Equalized learning rate

- Equalized learning rate
	- A technique used to ensure that all model parameters (weights) are updated at a similar speed during training, regardless of their position in the network or initial scale
	- Ensure that all the weights $w$ are normalized $w'$ to be within a certain range such that $w' = w/c$
	- Constant $c$, different for each layer, depending on the shape of the weight matrix
	- Ensures that parameters can take larger steps to reach optimum if needed
- Kerras
	- Uses a simple standard normal initialization and then scales the weights per layer at runtime
- Adam
	- Allows learning rates to be different for different parameters
	- Adjust the back-propagated gradient by the estimated standard deviation of the parameters, which ensures that the scale is independent of the update
	- Does not takes into account the dynamic range

```python
def equalized_learning_rate(shape, gain, fan_in=None):
	if fan_in is None: fan_in = np.prod(shape[:-1])
	std = gain / K.sqrt(fan_in) # He
	wscale = K.constant(std, name='wscale', dtype=np.float32)
	adjusted_weights = K.get_value('layer', shape=shape,
			intializer=tf.initializers.random_normal()) * wscale
	return adjusted_weights
```

- The default value is the produce of all the shape dimensions minus the feature map dimensions
	- Incoming connections per neuron
- He's initialization constant
- Creates a constant out of the adjustment
- Gets values for weights and uses broadcasting to apply the adjustment

- Restricting weight values between -1 and 1

## Pixel-wise feature normalization in the generator

![[Pasted image 20260309101324.png]]

- Batch normalization is too memory intensive for our resolution
- Pixel normalization
	- Uses activation magnitude at each layer before the input is fed into the next layer

![[Pasted image 20260309101424.png]]

![[Pasted image 20260309101440.png]]

**Pixel-wise Feature Normalization**
1. Take the pixel value of the feature map $(fm)$ as position (x, y)
2. Construct a vector for each (x, w)
	1. $V_{0, 0}$ = $[(0, 0) value for fm_1, (0, 0) value for fm_2 ...]$
	2. $V_{0, 1}$ = $[(0, 1) value for fm_1, (0, 1) value for fm_2 ...]$
	3. $V_{n, n}$ = $[(n, n) value for fm_1, (n, n) value for fm_2 ...]$
3. Normalize ach vector $v$ to have a unit norm, $n_{i, i}$
4. Pass that in the original tensor shape to the next layer

- This formula normalizes (divides by the expression under the squared root) each vector constructed
- Addition of a small noise term $(\epsilon)$
	- Not dividing by zero
- This term is applied only to the Generator
	- Explosion in activation magnitudes leads to an arms race only if both networks participate

```python
def pixelwise_feat_norm(inputs, **kwargs):
	normalization_constant = K.backend.sqrt(K.backend.mean(
		inputs**2, axis=-1, keepdims=True) + 1.0e-8)
	return inputs / normalization_constant
```

# TensorFlow Hub and Hands-On

```python
import matplotlib.pyplot as plt
import tensorflow as tf
import tensorflow_hub as hub

with tf.Graph().as_default():
	module = hub.Module("https://tfhub.dev/google/progan-128/1")
	latent_dim = 512
	latent_vector = tf.random_normal([1, latent_dim], seed=1337)
	interpolated_images = module(latent_vector)
	with tf.Session() as session:
	session.run(tf.global_variables_initializer())
	image_out = session.run(interpolated_images)
	
plt.imshow(image_out.reshape(128, 128, 3))
plt.show()
```
