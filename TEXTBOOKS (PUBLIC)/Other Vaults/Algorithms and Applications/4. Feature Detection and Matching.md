# Points and patches

- 2 main approaches for finding feature points and their correspondences
	1) Find feature in one image that can be tracked using a local search technique (nearby images)
		1) Correlation
		2) Least square
	2) Independently detect features

- Split the keypoint detection and matching pipeline into 4 separate stages

1. Feature detection
	1. Each image is searched for location that are likely to match well in other images
2. Feature description
	1. Each region around detected keypoint locations is converted into a more compact and stable descriptor
3. Feature matching
	1. Searches for likely matching candidates in other image
4. Feature tracking
	1. Searches a small neighbourhood around each detected feature
	2. Video processing

## Feature detectors

- Textureless patches are nearly impossible to localize
- Patches with large constrast changes (gradients) are easier to localize
	- Straight line at a single orientation suffer from aperture problem
- Patches with gradients in at least two different orientations are the easiest to localized (corner)

**Adaptive non-maximal suppression (ANMS)**

- Local maxima in the interest function, can lead to uneven distribution of feature points
	- Points will be denser in regions of higher constrast
- Only detect features that are both local maxima and whose response values is significantly greater then all of its neighbours within radius (10%)

![[Pasted image 20251110191639.png]]

**Measuring repeatability**

- Repeatability of feature detectors
- The frequency of key points detected in one image in corresponding location in a transformed image
- Transform planar images by applying rotations, scale changes, illumination changes, viewpoint changes, and noise
- Measure the information content available at each detected feature point
	- Define entropy as a set of rotationally invariant local grayscale descriptors

**Scale Invariance**
- Detecting features at the finest table scale may not be suitable
- Extract features at a variety of scales
- For most applications, the scale of the image is unknown
- Many scales are used for efficient feature extract

![[Pasted image 20251110192010.png]]

- Using extrema in the Laplacian of Gaussian (LoG) for interest points
- Harris operator, pixels where there is strong asymmetry in the local curvature

![[Pasted image 20251110192259.png]]

- Scale Invariant Feature Transform (SIFT) performs well, not based on the theoretical foundation of maximum spatial stability

**Rotational Invariance and Orientation Estimation**

- Most image matching and object recognition algorithms deal with in-plane image rotations

![[Pasted image 20251110192415.png]]

- A better method is to estimate a dominant orientation at each detected keypoint
- Once the location orientation and scale is estimated, a scaled and oriented patch around the points can be used
- Average gradient within a region around the keypoint
- Gaussian weighting function
	- First-order steerable filter
	- Use a large aggregation window
- Averaged gradient region can be small and unreliable of orientation
- Histogram is used around key points
- 36-bin histogram of edge orientations weighted by gradient magnitude and Gaussian distance to the centre


**Affine Invariance**

- Wide baseline stereo matching or location recognition, full affine invariance is preferred
- Affine detectors respond to deformations such as perspective foreshortening
- Ellipse to the autocorrelation or Hessian matrix
	- Eigenvalue analysis
	- Principal axes and ratios to fit affine frame
- Square root of the moment matrix can be used to transform local patches into a frame with similar rotation

![[Pasted image 20251110192815.png]]

- Maximally stable extreme region (MSER)
	- Affine coordinate frame can be fit to each detected region using moment matrix

## Feature descriptors

- For video sequences, features are mostly translational
- Sum of squared differences or normalized cross-correlation can be used to directly compare the intensities in small patches around feature points

**Bias and Gain Normalization (MOPS)**

- Do not exhibit large amount of foreshortening
	- Image stiching
- Multi-scale oriented patches (MOPS) are sampled are sampled from 5 pixels relative to detection scale

**Scale Invariance Feature Transform (SIFT)**

- SIFT features compute a gradient at each pixel around detected key point
	- Gaussian pyramid
- Magnitudes are down weighted by a Gaussian fall-off function
- Gradient histogram is formed

![[Pasted image 20251110193527.png]]

**PCA-SIFT**

- Simpler way to compute descriptors like SIFT

**Gradient Location-Orientation Histogram (GLOH)**

- Is a variant on SIFT that uses log-polar binning structure instead of four quadrants

![[Pasted image 20251110193652.png]]

**Steerable Filters**

- Combination of derivative of Gaussian filters

**Performance of Local Descriptors**

- GLOH performed best, followed by SIFT
- Develop class or instance-specific feature detectors that maximize discriminability from other classes

# Lines

## Successive Approximation

- Line simplification
	- Recursively subdivided the curve at the point furthest away from the line joining the two edge points
	- Used to approximate the original curve

## Hough transforms

- Each edge point votes for all possible lines passing through it
- Lines corresponding to high accumulator or bin values are fitted
- A hybrid strategy uses edgel votes for a number of possible orientation or location pairs centred around the estimate orientation

![[Pasted image 20251111102536.png]]

![[Pasted image 20251111102553.png]]

- An alternative representation uses a cube map
	- Projecting onto the face of a unit cube
	- All of the lines passing through a point correspond to line segments on the cube faces

![[Pasted image 20251111102644.png]]


**RANSAC-based Line Detection**

- Randomly chooses pairs of edgels to form a line hypothesis and tests other edgels to fall onto this line
- No accumulator array is needed
## Vanishing points

- Horizontal and vertical building edges, zebra crossings, railway tracks, and edges of furniture have the same vanishing points
	- Parallel in 3D
- Vanishing points help determine the intrinsic and extrinsic orientation of the camera
