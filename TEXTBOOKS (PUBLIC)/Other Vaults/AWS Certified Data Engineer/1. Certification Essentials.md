### What Is a Data Engineer?

- Data engineers are architects behind the scenes
- Critical infrastructure that powers all data solutions
- Creating high-performance data workflows that handle mass information with low latency and reliability
	- Robust ETL (Extract, Transform, Load) processes
	- Data warehouses and lakes
	- Real-time streaming solution
	- Data quality at scale
- Data processing
- SQL analytics
- Streaming
- Search
- Business intelligence

Services

- Amazon Kinesis

### Exam Topics
- Domain 1: Data Ingestion and Transformation (34%)
	- Data ingestion
	- Transform and process data
	- Data pipelines
	- Queries
- Domain 2: Data Store Management (26%)
	- Analyzing data with AWS services
	- Automating data processing
	- Maintaining and monitoring data pipelines
	- Ensuring data quality
- Domain 4: Data Security and Governance (18%)
	- Applying authentication and authorization mechanisms
	- Data encryption and masking
	- Preparing logs for auditing
	- Privacy and governance
