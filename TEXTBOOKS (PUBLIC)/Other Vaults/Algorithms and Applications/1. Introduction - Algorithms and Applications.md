# What is computer vision?

- Techniques for accurately computing a partial 3D model of an environment from partially overlapping photographs
- Given a large enough set of views of particular object of facade, we can create accurate dense 3D surface models using stereo matching
- Vision is an inverse problem, in which we seek to recover some unknowns given insufficient information to specify the solution
- Physic-based and probabilistic models help to disambiguate between potential solutions
- The forward models used in computer vision are developed in physics and in computer graphics
	- Radiometry
	- Optics
	- Sensor design
- In computer vision, images are reconstructed through properties
	- Shape
	- Illumination
	- Color distribution

- Important to have realistic test data, both synthetic, which can be used to verify correctness and analyze noise sensitivity, and real-world data typically of the way the system will finally be used
- Uses a statistical approach to formulating and solving computer vision
- The association of prior distributions with unknowns is often called Bayesian modeling
- Set inference algorithm to minimize the expected risk
- Gather training data to learn probabilistic models

# A brief History

- Early attempts at scene understanding involved extracting edges and then inferring the 3D structure of an object or a "blocks world" from the topological structure of the 2D lines

- 1970: Several line labeling algorithms were developed
- Generalized cylinders; solids of revolution and swept closed curves
	- Also called elastic arrangement of parts pictorial structures
	- Used in object recognition

<img src="/images/Pasted image 20251106123513.png" alt="image" width="500">

- David Marr introduced his notion of the 3 levels of description of a visual information processing system

1. Computational theory
2. Representation and algorithms
3. Hardware implementation

 - 1980: Image pyramids stated being widely used to perform tasks such as image blending and coarse-to-fine correspondence search
 - Scale-space processing were developed
 - Wavelets stated displacing or augmenting regular image pyramids in some applications
 - Use of stereo as quantitative shape cue was extended by a wide variety of shape-from-X techniques
	 - Shape from shading
	 - Photometric stereo
	 - Shape from texture
	 - Shape from focus
	- Stereo, flow, shape-from-X, and edge detection algorithms could be unified, or at least described, using the same mathematical framework id they were posed as variational optimization problems
- Contour trackers such as snakes
- 3D physically based models
- Such problems could equally well be formulated using discrete Markov Random Field (MRF) models
- 3D range data processing
	- Acquisistion
	- Merging
	- Modeling
	- Recognition
