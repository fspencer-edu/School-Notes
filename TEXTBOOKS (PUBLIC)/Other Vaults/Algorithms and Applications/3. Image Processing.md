## Morphology


- Thresholding enhances converts a grayscale image into a binary image
- Morphological operators
	- Change the shape of binary object
- Convolve the binary image with a binary structuring element
- Then select a binary output value depending on threshold result of convolution

Binary morphology operations

Dilation: $dilate(f, s) = \theta(c, 0)$
erosion: $erode(f, s) = \theta(c, S)$
majority: $maj(f, s) = \theta(c, S/2)$
opening: $open(f, s) = dilate(erode(f, s), s)$
closing: $close(f, s) = erode(dilate(f, s), s)$

- As dilation grows object consisting of 1s, while erosion shrinks
- Opening and closing operations leave large regions and smooth boundaries unaffected, while removing small object or holes

Issues
- Holes/gaps
	- Some destination pixels receive no value if no source pixel
	- Black spots
- Overlaps
	- Multiple source pixels may map to the same destinations
	- Brightness spike

- Splatting is a technique used in image warping, resampling, and 3D rendering to resolve holes or gaps in forward warping
	- Distributes each source pixel's intensity across multiple nearby destination pixels to fill holes and smooth transitions
	- Causes blurring and aliasing
- Inverse warping where each pixel in the destination image $g(x')$ is sampled from the original image $f(x)$
