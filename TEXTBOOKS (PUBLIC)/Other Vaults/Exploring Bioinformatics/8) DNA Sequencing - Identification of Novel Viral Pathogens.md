- Sequence assembly is one of the most complex bioinformatics problems
- Forced to rely on technical solutions to increase accuracy
	- Continuting to sequence random fragments
- Assembly program keeps track of coverage
- A common initial criterion is to choose largest overlap
- Longest possible overlap is one less than the length of the shortest sequence
- Start with max length

**Determining Largest Overlap Algorithm**
1. Start with two sequences
2. Set size n = size of smallest - 1
3. Compare n suffix from s1 and s2
4. Count matching bases in overlap region
5. If number of matches is less than n, subtract 1 from n
6. If n is 0, there is no overlap, repeated step 3

s1 = AATGCCTGA
s2 = TGACGAGTTAATGC

```c
AATGCCTGA
TGACGAGTTAATGC

        AATGCC TGA
TGACGAGTTAATGC

TGACGAGTTAATGCTGA
```

- Algorithm would first loop at 8-base overlaps
	- n = 9 - 1 = 8
- Match found at n = 5
- Therefor would merge to form the contig TGACGAGTTAATGCTGA
- Real sequence considers
	- Opposite strand
	- Errors
	- Millions of fragments
- Exhaustive matching of pairs of fragments will become impractical, so heuristics is used
- One heuristic solution is a "greedy" algorithm
	- Given the choice of overlapping fragment A with B, B with C, or C with A, program takes a "educated guess" for the largest overlap

## Sequencing DNA
- 3 commonly used sequencing platforms for analyzing sequences

**Automated Sanger Sequencing**
- Fred Sanger (1975)
- Rapid sequencing of long DNAs
- Human genome project
- Fluorescent nucleotides and automation
- Dideoxy sequencing uses DNA polymerase
	- Enzyme that replicates DNA in the cell
- A short single-stranded primer binds to the template and serves as the staring point
- In addition to dNTPs, dideoxy nucleotides are added
	- Synthesis stops when added
- Fragments are placed on a gel-like matrix in a tiny capillary tube and en electric current is applied
- DNA fragments (negatively charged) move to the positive pole, with smaller fragments moving faster
- Computer-connected reader determined which base the fragment ends with by the colour of the fluorescence
- Each succeeding fragment is one nucleotide longer, and colour and intensity allows the DNA "trace"

![[Pasted image 20251103120301.png]]


**Shotgun Sequencing**

- In directed sequencing, a primer is used to obtain sequence from a particular template, and then a new primer can be synthesized to match the just-read bases from the end of that sequence and the process is repeated
	- Process is slow
- Shotgun sequencing and computerized assembly
	- A long DNA is fragmented by mechanical shearing or enzymatic digestion into many short pieces, each joined to a cloning vector (plasmid)
- Sequence from each end of the fragment can be obtained using primers that match vector sequences
- Not necessary to wait for new primers to be synthesized
- Fragmenting and cloning can process at the same time
- Useful in next-generation sequencing methods


**454 Sequencing**
- Use in next-generation sequencing method
- Developed by 454 Life Sciences (2004)
- Pyroprocessing method
- DNA to be sequenced myst be fragmented
	- Chemically
	- Mechanically
	- Enzymatic digestion
- Made blunt (no single-stranded overhands) on each end
- Short oligonucleotide adapters of known sequence are then joined on each end of the fragments
- Single-stranded DNA with adapters become immobilized on the breads

![[Pasted image 20251103120739.png]]


- Immobilized fragments are amplified
- Beads are captured in individual oil droplets containing PCR reagents
- Primers matching the adapters are used to generate some 10 million copies of the original fragments
- A camera monitors each well and detects the light
- Indicating that a particular nucleotide was successfully added to the growing chain
- Process repeats with each of the other three nucleotides
