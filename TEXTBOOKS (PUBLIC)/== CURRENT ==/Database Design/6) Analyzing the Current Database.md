## Current Database

- Determine whether the database supports the organization's current information requirements
- Uncover existing structural deficiencies
- Determine how the database needs to evolve so that is will support the organization's future information requirements

- Analyzing an existing database
	- Paper-based databases
		- File systems
	- Legacy databases
	- Human knowledge bases

- Do not adopt the current structure as the basis for the new database structure

### Paper-Based Databases

- Incorporates data that is literally collected, stored, and maintained on paper
### Legacy Databases

- A database that has been in existence and in use for 5 years or more
- Individual who originally created the database either shifted responsibility or no longer at the organization
- Improperly or inefficiently designed
- Duplicate fields
- Redundant data
